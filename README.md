# Keyboard
## Steps
1. Go to the editor: https://nickcoutsos.github.io/keymap-editor/
2. Make changes you want. Check here for docs on what keys can do: https://v0-3-branch.zmk.dev/docs/keymaps/behaviors
3. Click **Save** and add a message
4. Go to the actions tab: https://github.com/v4climate/zmk-config/actions
5. Click on the latest save message
6. Scroll to the bottom and download the artifact
7. Open the the zip folder and custom files (listed in Names below)
8. Double press the reset button on the bottom of the keyboard
9. Copy file over to the LEFT keyboard
10. Let it reset and test
11. Repeat copy file for the RIGHT keyboard

## Uh oh
1. Double press the rest buttom on the bottom
2. Over the Reset File
3. Let it reset and test
4. Retry copying the files over
